• 安装各种环境

    1. OpenJDK8： yum -y install java-1.8.0-openjdk

    2. MySQL：CentOS默认软件列表中移除了MySQL，所以要先去官网 下个rpm包
      wget xxx
      rpm -ivh xxxxx.rpm
      yum update
      yum install mysql-server
      mysqld –initialize
      systemctl start mysqld
      结果启动服务的时候报错：Job for mysqld.service failed because the control process exited with error code. See “systemctl status mysqld.service” and “journalctl -xe” for details.
      根据提示看了下错误 Error: 2 (No such file or directory)，还是看日志吧，一般日志在 /var/log 目录下，报错如下：

      2018-11-12T13:13:00.297733Z 1 [ERROR] [MY-012271] [InnoDB] The innodb_system data file 'ibdata1' must be writable
2018-11-12T13:13:00.297766Z 1 [ERROR] [MY-012278] [InnoDB] The innodb_system data file 'ibdata1' must be writable
2018-11-12T13:13:00.297787Z 1 [ERROR] [MY-010334] [Server] Failed to initialize DD Storage Engine
2018-11-12T13:13:00.297904Z 0 [ERROR] [MY-010020] [Server] Data Dictionary initialization failed.
2018-11-12T13:13:00.297929Z 0 [ERROR] [MY-010119] [Server] Aborting

      看来是权限的问题，修改一下mysql下的权限 chmod -R 777 /var/lib/mysql，重新启动，正常了~

    3. 安装Nginx： yum -y install nginx ，同理启动相应服务： systemctl start nginx.service
      PS：配置文件所在目录： /etc/nginx/nginx.conf
